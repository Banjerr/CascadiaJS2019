# Controlling Drones with JavaScript

Using UDP (which is apparently not a great protocol for this kind of thing), this developer was able to control a drone/swarm of drones with JavaScript, using their telemetry/camera and patterns on the flooor to determine their position and facilitiate their movement with each other in a syncopated style.
